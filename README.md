# hubot-lot

lot

See [`src/lot.coffee`](src/lot.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-lot --save`

Then add **hubot-lot** to your `external-scripts.json`:

```json
[
  "hubot-lot"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
